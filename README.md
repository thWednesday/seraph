# seraph

Seraph is a hash bruteforce program

# Installation

```bash
git clone https://github.com/thWednesday/seraph
cd seraph
cargo install --path .
```

## Usage

```bash
seraph -h
```

## Features

- Hash bruteforcing
  - Hash type guessing
  - Multiple hash type support
    - MD5
    - SHA2-SHA224
    - SHA2-SHA256
    - SHA2-SHA384
    - SHA2-SHA512
- Custom alphabet
  - Alphabet presets
  - Maximum hash length
